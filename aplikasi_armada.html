Konfigurasi Firebase (Sinkronisasi Cloud)

Hubungkan ke Firebase Realtime Database agar data tersinkronisasi ke semua device. Buka Firebase Console

Langkah Setup:
  1. Buka Firebase Console → Create Project
  2. Build → Realtime Database → Create Database → asia-southeast1 → Start in test mode
  3. Project Settings → Your apps → Web app → Register → Salin firebaseConfig
  4. Isi nilai di bawah, klik Simpan & Hubungkan
